This is a pixel-perfect responsive replica of the official homepage of SR Group of Institutions, Jhansi (srgi.ac.in), built as part of a frontend internship assignment for ACTT Connect.
The project replicates the complete homepage layout including all major sections, styling, and interactive functionality using only pure HTML, CSS, and JavaScript — no frameworks or libraries used.

Sections Included

Top Info Bar — Contact info, email, social media links
Sticky Header — Logo, toll-free number, Apply Now button
Navigation Bar — Fully responsive with hamburger menu for mobile
Hero Slider — Auto-playing image slider with 3 slides, prev/next buttons and dot navigation
Notice Ticker — Auto-scrolling announcement marquee
Stats Bar — Key numbers like students enrolled, faculty count, companies
About Us — Institution overview with key highlights
Programs/Courses — 6 course cards covering B.Tech, MBA, MCA, Diploma, Ph.D.
Why Choose SRGI — 8 feature cards highlighting strengths
Notice Board — Latest announcements with dates and Quick Links panel
Alumni Testimonials — 3 student reviews with ratings
Admission CTA — Call-to-action banner for 2024-25 admissions
Campus Gallery — Responsive image grid with hover overlay effect
Footer — 4-column footer with links, contact info, and social icons


Key Features

Fully responsive design — works on mobile, tablet, and desktop
Mobile hamburger menu for navigation on small screens
Auto-playing hero slider with manual controls and dot indicators
Scrolling notice ticker for announcements
Hover effects on cards, gallery, and navigation links
CSS custom properties (variables) used for consistent theming
CSS Grid and Flexbox used for all layouts
Clean, well-commented and organized code structure
